PPC.IO · SURFACE TAXONOMY

Moments, not modes.

Four surface types. Each gets one treatment. No theme toggle, no exceptions.

01 · ALWAYS DARK

Navigation rail

Rule: the sidebar is the only persistent dark element. It anchors the brand.

Applies to: sidebar, top-bar if you ever add one, mobile bottom nav.

02 · LIGHT + DARK HERO

Landing and decision pages

Rule: dark hero block sits inside light canvas. Hero gets the headline + primary action. Cards underneath stay white.

Applies to: agents library (Image 1), project overview, dashboard home, account creation.

03 · LIGHT + DARK DRAMA

Live moments and events

Rule: entire content area goes dark for live/cinematic states. Use giant headlines. Progress, not data tables.

Applies to: agent loading (Image 4), agent running, mission control live view, success states.

04 · PURE LIGHT

Work surfaces

Rule: white cards on lavender. Status communicated through 3px left-border strips, never full backgrounds.

Applies to: projects table (Image 2), findings, audit results, settings, AI instructions, business context, reports.

FOR YOUR AI DESIGN AGENT · COPY THIS

PPC.io uses a "moments not modes" design system. Sidebar is always dark (#0F0A1E). Main canvas is always lavender (#ECEAFA), never pure white. Work surfaces use white cards (#FFFFFF) with 0.5px solid #d9d4ec borders. Status communicated via 3px left-border strips, never full-color backgrounds. Decision pages use dark hero blocks inside the lavender canvas. Live agent states take over the full content area with dark backgrounds + giant headlines. Bright purple (#7F5AF0) is used once per screen, max, for the top recommendation or primary CTA. There is no dark mode toggle. Typography: large bold headlines for moments, regular weight for work. Trailing purple period (.) is a brand motif on headings.
